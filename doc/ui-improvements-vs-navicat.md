# iDBLink UI/交互完善文档（对照 Navicat Premium）

> 本文档用于指导其他 AI 进行开发，详细列出当前 iDBLink 与 Navicat Premium 在 UI 和交互层面的差距，按模块分类，每个条目包含「Navicat 标准做法」「当前状态」「需完善点」「优先级」。
> **文档版本**: v2.1 | **更新日期**: 2026-04-28 | **对照基准**: Navicat Premium 16/17
>
> **本轮更新**: 完成事务控制、SQL编辑器增强、大文本编辑、列头筛选、ER图列详情、窗口标题动态更新、Excel导出、导入向导等 14 项功能。
