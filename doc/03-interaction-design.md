# iDBLink - 交互操作设计文档

## 概述

本文档详细描述 iDBLink 的用户交互操作流程，仿照 Navicat Premium 的操作习惯，降低老用户学习成本。所有交互设计均基于 [UI 设计文档](./02-ui-design.md) 的界面布局。

---

### 1.1 冷启动流程

```
启动应用 → 加载上次会话状态 → 渲染主界面
├── 恢复上次打开的标签页（SQL 查询、数据表浏览）
├── 恢复左侧面板折叠/展开状态
├── 恢复底部日志面板高度
```

## 二、连接管理交互

### 2.1 新建连接流程

**触发方式：**
- 工具栏 [➕ 新建连接]
- 左侧面板右键 → 新建连接
- 菜单栏 文件 → 新建连接
- 快捷键 `Ctrl/Cmd + N`

**交互流程：**

```
1. 弹出连接配置对话框（模态）
2. 顶部选择数据库类型标签（MySQL/PostgreSQL/SQLite/...）
3. 填写连接信息：
   ├── 连接名称（自动生成：新 MySQL 连接）
   ├── 主机地址（默认 localhost）
   ├── 端口（按数据库类型预填）
   ├── 用户名
   ├── 密码（支持显示/隐藏切换）
   └── 初始数据库（可选）
4. 点击 [测试连接] → 显示成功/失败反馈
5. 点击 [保存连接] → 左侧树新增节点 → 对话框关闭
```

**连接配置对话框交互细节：**

| 交互 | 行为 |
|------|------|
| 切换数据库类型 | 端口自动变更（MySQL:3306, PG:5432, SQLite:文件选择） |
| 输入主机地址 | 实时验证 IP/域名格式 |
| 输入端口 | 限制 1-65535 范围 |
| 密码字段 | 默认隐藏，点击 👁 切换明文 |
| 测试连接 | 按钮显示 loading，结果用 Toast 提示 |
| 保存连接 | 验证必填项，失败时高亮错误字段 |
| SQLite 类型 | 主机/端口/用户名隐藏，替换为"数据库文件"路径选择 |

**Tab 页签（高级配置）：**

```
[常规] [SSL] [SSH 隧道] [高级]

常规：基础连接信息
SSL：
  ├── ☑ 使用 SSL
  ├── SSL 模式：[REQUIRED / VERIFY_CA / VERIFY_FULL]
  ├── 客户端证书路径
  ├── 客户端密钥路径
  └── CA 证书路径

SSH 隧道：
  ├── ☑ 使用 SSH 隧道
  ├── SSH 主机
  ├── SSH 端口（默认 22）
  ├── SSH 用户名
  ├── 认证方式：[密码 / 私钥]
  └── 私钥路径（选择私钥时显示）

高级：
  ├── 连接超时（秒，默认 30）
  ├── 字符集：[UTF-8 / GBK / ...]
  ├── 时区设置
  ├── 连接池大小（默认 5）
  └── 自定义连接参数（键值对）
```

### 2.2 连接/断开连接

**连接操作：**

| 操作 | 触发方式 | 行为 |
|------|----------|------|
| 单击未连接节点 | 鼠标左键 | 自动尝试连接 |
| 双击连接节点 | 鼠标左键 | 连接并展开数据库列表 |
| 右键 → 连接 | 上下文菜单 | 尝试连接 |

**连接中状态：**
- 节点显示 loading spinner
- 状态栏显示"正在连接到 xxx..."
- 连接失败时弹出错误提示，节点恢复未连接状态
- 连接成功后节点显示绿色圆点，自动展开数据库列表

**已连接状态：**
- 节点旁显示绿色圆点 ●
- 自动加载并显示数据库列表
- 状态栏更新："✓ 已连接：xxx | xx ms"
- 右键菜单变为：断开连接、刷新、编辑连接...

**断开连接：**
- 右键 → 断开连接
- 点击节点旁的断开图标
- 关闭所有该连接相关的打开标签页（有未保存内容时需确认）

### 2.3 连接右键菜单

**未连接节点：**
```
├── 连接
├── 编辑连接
├── 复制连接配置
├── ─────────
├── 移动至分组 → [子菜单：分组列表]
└── 删除连接（需确认）
```

**已连接节点：**
```
├── 断开连接
├── 刷新
├── 编辑连接
├── 复制连接配置
├── ─────────
├── 新建查询
├── 命令列界面
├── ─────────
├── 转储 SQL 文件 → [仅结构 / 结构和数据]
├── 运行 SQL 文件
├── ─────────
├── 移动至分组 → [子菜单]
└── 删除连接（需确认，需先断开）
```

### 2.4 连接编辑

- 右键 → 编辑连接，打开连接配置对话框（预填现有信息）
- 修改后需重新测试连接
- 保存时如果连接正在使用，提示"连接正在使用中，修改将在下次连接时生效"

### 2.5 复制连接配置

- 右键 → 复制连接配置
- 生成新连接，名称自动添加"副本"后缀
- 密码不复制（需重新输入），其他配置完全一致

## 三、自定义分组交互

### 3.1 创建分组

**触发方式：**
- 工具栏 [📁 新建分组]
- 左侧面板空白处右键 → 新建分组
- 分组节点右键 → 新建分组

**分组配置对话框：**
```
┌─────────────────────────────┐
│     新建分组                │
├─────────────────────────────┤
│ 分组名称                    │
│ [新分组____________]        │
│                             │
│ 分组图标                    │
│ [📁] [🌐] [💻] [🗄️] ...   │
│                             │
│ 分组颜色                    │
│ [🔵] [🟢] [🟠] [🔴] ...   │
│                             │
│      [取消]  [创建分组]     │
└─────────────────────────────┘
```

### 3.2 分组操作

| 操作 | 触发方式 | 行为 |
|------|----------|------|
| 重命名 | F2 / 右键 → 重命名 | 节点名称变为可编辑，Enter 确认，Esc 取消 |
| 删除 | 右键 → 删除分组 | 弹出确认"删除分组及其下的所有连接？" |
| 更改图标 | 右键 → 更改图标 | 弹出图标选择器 |
| 更改颜色 | 右键 → 更改颜色 | 弹出颜色选择器 |
| 展开/折叠 | 点击箭头 / 双击 | 切换展开状态，状态自动保存 |
| 导入分组 | 右键 → 导入分组 | 选择 .idbl 文件导入 |
| 导出分组 | 右键 → 导出分组 | 导出分组配置为 .idbl 文件 |

### 3.3 拖拽操作

- **连接 → 分组**：将连接移动到目标分组
- **连接 → 连接**：调整连接排序
- **分组 → 分组**：调整分组排序
- 拖拽时显示目标高亮，松开后完成移动
- 拖拽操作不可撤销（可通过撤销操作恢复）

### 3.4 分组右键菜单

```
├── ➕ 新建连接
├── ➕ 新建分组
├── 🔄 刷新
├── ─────────
├── ✏️ 重命名 (F2)
├── 🎨 更改图标
├── 🎨 更改颜色
├── ─────────
├── 📤 导出分组
├── 📥 导入分组
└── 🗑 删除分组（需确认）
```

## 四、数据库对象树交互

### 4.1 连接成功后的对象加载

```
连接成功 → 自动获取数据库列表 → 展开显示
├── 📋 表 (12)
```

```
├── 🔄 刷新数据库
├── 📝 新建查询

├── 📤 转储 SQL 文件 → [仅结构 / 结构和数据]
```

**表节点右键菜单：**
```
├── 打开表（浏览数据）
```

```
├── 打开视图（浏览数据，只读）
├── 设计视图（查看/编辑视图定义 SQL）

└── 属性
```

```
├── 执行（弹出参数输入对话框）
├── 编辑（打开源代码编辑器）
├── 重命名
├── 删除（需确认）
├── ─────────
└── 属性
```

### 4.6 对象树搜索过滤

**左侧面板顶部搜索框：**
- 输入关键词 → 实时过滤树节点（300ms 防抖）
- 匹配规则：表名、视图名模糊匹配
- 清除按钮（×）一键重置
- 无匹配结果时显示"未找到匹配的对象"
- 搜索范围：当前展开的连接/数据库内

**数据编辑交互细节：**

**底部工具栏详细交互：**

**中间 SQL 显示区：**
- 显示当前数据对应的 SQL 语句
- 可点击编辑（编辑后变为自定义查询）
- 右侧 [▶ 执行] 按钮
- [📋 复制 SQL] 按钮

**右侧分页区：**
- 页大小选择：[10 ▼] [50] [100] [500]
- 翻页控件：[首页] [上一页] [1] [2] [3] [...] [下一页] [末页]
- 跳转输入框：[跳转至 __ 页] [GO]
- 统计信息：显示"第 1-50 行，共 1,234 行"

**子 tab 切换：** `[浏览] [设计] [信息] [权限]`

**布局结构：**
```

│ 字段定义表格：                                           │
│ ┌───┬─────────┬──────────┬──────┬──────┬──────┬────────┐│
│ │PK │ 字段名  │ 类型     │ 长度 │ 小数 │ 默认 │ 注释   ││
│ ├───┼─────────┼──────────┼──────┼──────┼──────┼────────┤│
│ │🔑 │ id      │ INT      │ 11   │      │ AUTO │ 主键   ││
│ │   │ username│ VARCHAR  │ 50   │      │      │ 用户名 ││
│ │   │ email   │ VARCHAR  │ 100  │      │      │ 邮箱   ││
│ │   │ status  │ TINYINT  │ 1    │      │ 0    │ 状态   ││
│ │   │ created │ DATETIME │      │      │ NOW  │ 创建时间││
│ └───┴─────────┴──────────┴──────┴──────┴──────┴────────┘│
├─────────────────────────────────────────────────────────┤
│ 底部子面板（可切换）：                                    │
│ [字段] [索引] [外键] [触发器]                            │
│                                                         │
│ 索引列表示例：                                           │
│ 名称        │ 类型   │ 字段          │ 注释              │
│ PRIMARY     │ UNIQUE │ id            │                   │
│ idx_email   │ NORMAL │ email         │ 邮箱索引          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 十七、异常处理交互

### 17.1 连接异常

| 异常 | 处理 |
|------|------|
| 连接超时 | Toast 提示"连接超时，请检查网络"，节点恢复未连接状态 |
| 认证失败 | Toast 提示"用户名或密码错误"，打开编辑连接对话框 |
| 连接断开 | 状态栏显示红色"已断开"，弹出 reconnect 提示 |
| 自动重连失败 | 通知用户"自动重连失败，请手动连接" |

### 17.2 查询异常

| 异常 | 处理 |
|------|------|
| SQL 语法错误 | 结果面板显示错误信息，编辑器中高亮错误位置 |
| 查询超时 | 自动停止，提示"查询超时，已自动停止" |
| 结果集过大 | 提示"结果超过 10000 行，已截断显示" |
| 权限不足 | 提示"权限不足，无法执行此操作" |

### 17.3 数据编辑异常

| 异常 | 处理 |
|------|------|
| 类型不匹配 | 单元格标红，提示"值类型不匹配" |
| 约束冲突 | 提交时提示"违反唯一约束/外键约束" |
| 并发冲突 | 提示"数据已被其他用户修改，请刷新后重试" |
| 提交失败 | 保留编辑状态，提示"提交失败：xxx" |

## 十八、多数据库类型差异交互

### 18.1 连接配置差异

| 数据库 | 特殊配置 |
|--------|----------|
| MySQL | 字符集、SSL 模式 |
| PostgreSQL | Schema 选择、SSL 模式 |
| SQLite | 数据库文件路径（代替主机/端口） |
| SQL Server | 实例名、认证方式（Windows/SQL） |
| Oracle | SID/Service Name、连接类型 |
| 达梦 | 端口（默认 5236）、字符集 |

### 18.2 对象类型差异

| 数据库 | 特有对象 |
|--------|----------|
| PostgreSQL | Schema、Sequence、Materialized View |
| Oracle | Package、Synonym、DB Link |
| SQL Server | Schema、Job、Linked Server |
| MySQL | Event、User |
| 达梦 | Package、DB Link |

### 18.3 数据类型显示差异

| 类型 | MySQL | PostgreSQL | SQLite |
|------|-------|------------|--------|
| 布尔 | TINYINT(1) | BOOLEAN | INTEGER |
| JSON | JSON | JSONB | TEXT |
| 数组 | - | ARRAY | - |
| 自增 | AUTO_INCREMENT | SERIAL | AUTOINCREMENT |
