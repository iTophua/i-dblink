## 🎯 项目目标

开发一款类似 Navicat Premium 的跨平台数据库管理工具：

- **支持数据库**：MySQL, PostgreSQL, SQLite, SQL Server, Oracle, MariaDB + 达梦数据库
- **目标平台**：Windows, macOS, Linux
- **技术栈**：Tauri v2 + Rust + React + TypeScript
- **界面风格**：Navicat Premium 经典三栏布局
- **核心特性**：自定义分组管理、智能 SQL 辅助、可视化 ER 图

## 🛠 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 应用框架 | Tauri v2 | 跨平台桌面框架，体积 ~10MB |
| 后端语言 | Rust | 高性能、安全 |
| 前端框架 | React 18 + TypeScript | 现代化 UI 开发 |
| UI 组件库 | Ant Design 5 | 丰富的企业级组件 |
| SQL 编辑器 | Monaco Editor | VS Code 同款编辑器 |
| 状态管理 | Zustand | 轻量级状态管理（< 1KB） |
| 数据网格 | AG Grid Community | 高性能虚拟表格 |
| 数据库驱动 | sqlx / tiberius / odbc-api | 异步 SQL 库 |
| 密钥存储 | keyring | 系统原生密钥链 |
| 异步运行时 | Tokio | Rust 事实标准 |
